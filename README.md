## Requirment

1. Required Dependencies

   - axios → for request API
   - chai → for assertion (result validation)

   > npm install --save-dev wdio-intercept-service chai axios dotenv

2. Make **.env** file and add **GOREST_TOKEN="your_token"**

3. Run the code:
   > npx wdio
